## ROS명령어
### roscore
- ROS Master를 실행한다
-노드를 켜기 전에 가장먼저실행
'''
    bash
     roscore
'''

### rosrun
-노드를 실행한다.
-rosrun 패키지이름 노드이름
'''
    bash
     rosrun <패키지이름> <노드이름>
'''
